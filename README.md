# aem-runner-bash
Bash Script to startup development environment on a mac

## Installation
* Pull this repo
* Reference it in your ./.bash_profile file
```
declare load_files=(~/personal/aem-runner-bash/aem-project-loader.bash)
```
* Configure AEM/repo vars
* Restart terminal
